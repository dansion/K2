Cdselector组件文档

小组件们

cdselector根据展现及功能的不同，包括：

  • 1、k2-cdselector

    普通的联动组件(select)

  • 2、k2-multiSelector

    最后一级可多选的联动组件(最后一级是checkbox)

  • 3、k2-cdselector-option

    与最初级的select类似，只是表现层不同(以ul，li的形式展现)

  • 4、k2-mapRegionSelectorNew

    地图用到的select

  • 5、k2-cdselector-optionByStep

    以li形式展现的联动组件，同时是否显示下一个ul，由父级决定，父级友子孩子则显示

  • 6、k2-mulitSelector-option

    可多选的联动组件，以li显示展现

调用方法

YUI().use("k2-xuanwuCode","k2-cdselector",function(Y){
   /*container表示容器的ID*/
  var cdselector_a=new Y.Cdselector({
    containerId:"container",
    config:{
     /*渲染模式，ul or select*/
    renderType:1,
    /*显示的级联数*/
    levelNum:4,
    /*自动生成的jsonname 即input隐藏域的name值*/
    jsonName:"cdselectorJson",
    /*是否有不限选项*/
    limitArray:[false,false,false,true],
    /*改变选择值后的处理方法*/
    afterChange:function(lv,s){
       Y.one("#container-vl").setContent("aa="+s.join("-"));
    },
    dataSource:Y.Cdselector_Base.xuanwuCode
     }
});
  cdselector_a.select("3300000000-3301000000-3301020000-3301020006");
});

add注释：

  • Y.Cdselector是k2-cdselector的构造函数
  • Y.MultiSelector是k2-multiSelector的构造函数
  • Y.Cdselector_Option是k2-cdselector-option的构造函数
  • Y.MapRegionSelectorNew是k2-mapRegionSelectorNew的构造函数
  • Y.Cdselector_OptionByStep是k2-cdselector-optionByStep的构造函数
  • Y.MulitSelector_Option是k2-mulitSelector-option的构造函数

各参数配置

参数名 说明 可选值
containerId 包含联动组件的容器id 必须
config 配置属性的对象 必须
config.levelNum 共有的层级数 默认为2
config.jsonName 当选级变化时，用改变的隐藏域的name值
config.limitArray 是否有不限选项的出现,true表示出现不限改选项 默认false
config.afterChange 联动后出发的事件lv表示层级，s表示联动后选中的级值 可选择使用
config.dataSource 设置数据源(数据源格式见下方) 必须
select 用于初始化的所选层级 可选择调用

现有的数据源

现有的数据对象有如下几种，同时也可自己配置数据源

  • Y.Cdselector_Base.xuanwuCode ：玄武编码的地域信息
  • Y.Cdselector_Base.categoryData ：分类信息数据源
  • Y.Cdselector_Base.regionCode ：老编码的地域信息

数据源格式

  • 一级: { "level":{}, "data":[] }
  • level级: "level":{0:"不限",1:"人类",2:"男人",3:"女人",4:"男孩",5:"妇女",6:"美女",7:"丑女"}
  • level级: "data":[ {0:[1]}, {1:[2,3]}, {2:{4},3:{5,6,7}} ]
