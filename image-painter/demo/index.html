K2 Image Painter docs

简介

K2 Image Painter组件用于在页面中绘制png 32的图片。它在IE下使用VML来渲染图片，而在其它浏览器使用SVG来渲染图片。

版本

  1. 最新版本 1.0.0

参数，事件和方法

构造函数参数
参数名 参数类型 参数说明 默认值
参数1：id String DOM节点的ID null
参数2：width Number 渲染区域的宽度 null
参数3：height Number 渲染区域的高度 null
方法
方法 说明
image(src:String, x:Number, y:Number, width:Number, height:Number):void 在特定的区域渲染某个图片

示例

YUI().use('k2-image-painter', function(Y){
	var painter = new Y.ImagePainter("painter", 200, 120);
	painter.image("apple.png", 50, 0, 100, 100);
});
