K2 Graphic docs

简介

Graphic组件用来在Web上绘制图形，为了兼容各个浏览器，它在高级浏览器上使用SVG, Canvas API，IE6-8上使用VML绘制。

API

lineStyle(stroke, lineColor, lineAlpha, ...)
设置线条色
beginFill(fillColor, fillAlpha)
设置填充色
moveTo(x, y)
重新移动画笔到某点
lineTo(x, y)
绘制直线到某点
curveTo(controlX1, controlY1, controlX2, controlY2, targetX, targetY)
绘制曲线
drawCircle(x, y, radius)
绘制圆形
drawRect(x, y, width, height)
绘制矩形
drawRoundRect(x, y, width, height, xRadius, yRadius)
绘制圆角矩形
drawEllipse(x, y, width, height)
绘制椭圆

YUI().use('k2-graphic', function(Y){
	var g = new Y.Graphic();
	g.lineStyle(2, '#ff0000', 0.5);
	g.beginFill('#ff6600', 0.8);
	g.drawCircle(50, 50, 30);
	g.render('#myGraphic');
});
